Social Engineer toolkit

Herramienta de ingeniería social con clonación de páginas web, phising, etc...

Conocido también por su abreviación SET, es una completísima suite dedicada a la ingeniería social, que nos permite automatizar tareas que van desde el envío de SMS(mensajes de texto)falsos, con los que podemos suplantar el número telefónico que envían el mensaje, a clonar cualquier página web y poner en marcha un servidor para hacer phising en cuestión de segundos

SET es multiplataforma y sólo necesitamos tener instalado el interprete de Python para ejecutarlo e iniciarlo con ./set o Python set en la carpeta donde decidamos guardarlo. SET viene integrado en Kali Linux pero lo podéis descargar directamente desde la pagina trustedsec, si queréis usar cualquier otra distribución . https://www.trustedsec.com/downloads/social-engineer-toolkit/ por medio de svn: svn co http://svn.trustedsec.com/social_engineering_toolkit set/

Bash 
$ setoolkit
